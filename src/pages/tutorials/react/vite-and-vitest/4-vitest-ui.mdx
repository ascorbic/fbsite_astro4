[Vite](https://vitejs.dev/)  offers a super useful package to manage your test by using a Web UI:

<Code>
```
npm i -D @vitest/ui
```
</Code>

You can now run from command line the following command:

<Code>
```
vitest --ui
```
</Code>

or add a new script to `package.json` to run it by using `npm run test:ui`.

It should automatically open a new page on your browser after running it:

<Code title="<Code>package.json</Code>" techs={['js']}>
```json
scripts: {
  // ...
 "test:ui": "vitest --ui",
}
```
</Code>
